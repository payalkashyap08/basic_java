# Basic Java Programs

## 📌 Description
This repository contains basic Java programs for beginners to understand core logic building.

## 🚀 Programs Included
- Taking user input
- Factorial of a number
- Prime number check
- Reverse a number
- Palindrome check
- Sum of digits
- Even or Odd check

## 🛠️ Concepts Used
- Loops (for, while)
- Conditional statements (if-else)
- Scanner class (input)
- Basic number logic

## 📌 Example
Enter a number: 5  
Factorial of 5 is: 120  
5 is a Prime Number  
Reversed number: 5  
5 is a Palindrome  
Sum of digits: 5  
5 is Odd  

## 🎯 Purpose
This project is useful for:
- Beginners learning Java
- Understanding logic building
